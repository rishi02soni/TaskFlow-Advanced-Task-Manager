# 🚀 TaskFlow – Advanced Task Manager

A modern, fully functional **Task Management Web App** built using **React.js**.
Designed with clean architecture, reusable components, and real-world features — perfect for **SDE-level projects & resumes**.

---

## 🌟 Features

* ✅ Add, Edit, Delete Tasks
* 🔄 Mark Tasks as Completed / Pending
* 🔍 Search Tasks in Real-Time
* 🎯 Filter Tasks (All / Completed / Pending)
* 💾 Persistent Storage using LocalStorage
* ⚡ Fast & Responsive UI
* 🧩 Component-Based Architecture

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Hooks)
* **State Management:** useState, useEffect
* **Unique IDs:** UUID
* **Styling:** CSS (can be upgraded to Tailwind)
* **Storage:** Browser LocalStorage

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/taskflow.git
```

### 2️⃣ Navigate to project folder

```
cd taskflow
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start the development server

```
npm start
```

## 🧠 How It Works

* Tasks are stored in **React state**
* Changes are synced with **LocalStorage**
* Filtering & searching are handled dynamically
* Each task has a unique ID for efficient updates

---

## 🔑 Key Concepts Used

* React Functional Components
* Hooks (useState, useEffect)
* Lifting State Up
* Conditional Rendering
* Controlled Components
* Array Methods (map, filter)
